## 규칙

- 각 섹션 혹은 예제 별로 브랜치로 나누어 관리한다.
- 각 브랜치 별로 DB 명을 다르게 사용한다
  - 계속 동일한 DB를 사용하면 데이터가 꼬이거나 테이블에 문제 발생함 :(
